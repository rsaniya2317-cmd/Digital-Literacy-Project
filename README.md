Project Overview
This digital literacy project focuses on developing essential skills required in today's world. Through this project, I learned the importance of creating an online presence, communicating professionally, and staying safe in the digital environment. I explored different platforms, which increased my confidence in using technology effectively. Each task helped me become more aware and responsible while using online tools.

Project Structure
Task 1: Infographic
In this task, I created an infographic to visually represent a concept related to digital literacy.
Task 2: Portfolio
This task involved creating my online profiles on different platforms. I learned how to build a professional digital identity and manage my online presence.
Task 3: Platforms
In this task, I explored platforms like HackerRank and Google Forms. HackerRank helped me practice coding, while Google Forms helped me learn how to create forms and analyze responses.
Task 4: Email Etiquette
This task helped me understand how to write emails in a professional format. I also created a checklist of do's and don'ts of social media.
Task 5: Cybercrime Case Study
In this task, I wrote a case study on UPI/online payment fraud and created a checklist to spread awareness about online safety and prevention methods.
